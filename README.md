TRADEO — BlindBox Intercambio

Aplicación web para la compra, venta e intercambio de figuras blind box entre coleccionistas. Proyecto final del Ciclo Superior de Desarrollo de Aplicaciones Web.

Tecnologías



Java 17 + Servlets + JSP (patrón MVC)

MySQL 8 (puerto 3307)

Apache Tomcat 9

HTML5, CSS3, JavaScript



Funcionalidades



Registro y login con gestión de sesiones y roles

Publicación de figuras con desplegables encadenados por marca, modelo y serie

Catálogo con filtros de búsqueda

Sistema de mensajería pública por figura

Perfil de usuario con edición de datos

Política de cookies conforme al RGPD



Requisitos para ejecutar



Java 17

Apache Tomcat 9

MySQL en puerto 3307

Crear base de datos blindboxintercambio con el schema de sql/schema.sql

Insertar datos iniciales con sql/popmart\_datos\_iniciales.sql

Configurar credenciales en src/java/com/blindboxintercambio/util/ConexionDB.java
